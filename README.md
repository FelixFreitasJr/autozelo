# AutoZelo

AutoZelo é um sistema desenvolvido em Java para gerenciamento de manutenção preventiva e corretiva de veículos.

O objetivo do projeto é permitir que proprietários acompanhem todo o histórico do veículo, registrando manutenções, despesas, abastecimentos e documentos importantes.

## Objetivos

- Cadastro de veículos
- Histórico de manutenções
- Controle de quilometragem
- Alertas de manutenção
- Controle de despesas
- Relatórios

## Tecnologias

- Java
- Git
- GitHub
